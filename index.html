🎓 Student Management System

Student Management System is a simple, responsive web application built with PHP and MySQL. It allows administrators to manage student records with features such as registration, editing, deleting, and searching. Perfect for small institutions or for learning basic CRUD operations.

🚀 Features

  • ✅ Register new students
  • 📄 View all student records in a dynamic table
  • ✏️ Edit student information
  • 🗑️ Delete students from the database
  • 🔍 Live search and sorting functionality
  • 📱 Fully responsive interface using Bootstrap

🛠️ Tech Stack

  • Frontend: HTML5, CSS3, Bootstrap 5
  • Backend: PHP
  • Database: MySQL

📂 Folder Structure

/student-management-system
│
├── index.php              # Dashboard with student list
├── add_student.php        # Form to register a new student
├── edit_student.php       # Form and logic to update student info
├── delete_student.php     # Handles record deletion
├── config.php             # Database connection settings
├── assets/                # CSS, JS, and images
└── README.html            # Project documentation

⚙️ Setup Instructions

  1. Clone the Repository
    git clone https://github.com/your-username/student-management-system.git
  2. Create a Database and import the structure: 
    CREATE TABLE `students` (
  `id` INT NOT NULL AUTO_INCREMENT,
  `name` VARCHAR(100) NOT NULL,
  `email` VARCHAR(100) NOT NULL,
  `course` VARCHAR(100) NOT NULL,
  `registration_date` DATE NOT NULL,
  PRIMARY KEY (`id`)
);
  3. Update Database Credentials in config.php: 
    $conn = new mysqli("localhost", "root", "", "student_db");
  4. Run the Project
    Place the project folder in your htdocs directory (if using XAMPP), and visit:
    http://localhost/student-management-system/
